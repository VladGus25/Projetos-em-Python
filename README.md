# Projetos-em-Python

Lista de Exercícios em Python

Este repositório contém dois arquivos com exercícios básicos de Python, focados em estruturas de dados e entrada de dados do usuário.

Arquivos

* `exercicios_python.py` → Exercícios com listas, média, entrada de dados, dicionários e repetição
* `exercicios2_python.py` → Exercícios com tuplas, dicionários, listas, sets e mini-sistemas

Conteúdos Trabalhados

Listas (`list`)
Tuplas (`tuple`)
Dicionários (`dict`)
Conjuntos (`set`)
Entrada de dados (`input`)
Estruturas de repetição (`for`, `while`)
Manipulação de dados

Como Executar

1. Instale o Python (se ainda não tiver)
2. Baixe os arquivos `.py`
3. No terminal, execute:

```bash
python exercicios_python.py
```

ou

```bash
python exercicios2_python.py
```
O que você vai encontrar no projeto:

Maior e menor valor de uma lista
Média e soma de valores
Manipulação de listas e dicionários
Cadastro de pessoas e funcionários
Remoção de duplicatas
Contagem de elementos
Simulação de sistemas simples

Objetivo
Praticar lógica de programação e aprender as principais estruturas do Python de forma simples e direta.
